## Intro

Microservice_A: Exposes an HTTP Restful API, that allows performing CRUD operations for the management of Fleet entities (i.e: Driver, Car, Trip, etc). 
Also provides endpoints for assigning a Driver to a Car

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
